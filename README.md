## Getting Started

This project uses [code generation tools](https://pub.dev/packages/build_runner).<br>
**Before building the project, run these commands:**<br>

```shell script
flutter clean
```

```shell script
flutter pub get
```

```shell script
flutter pub run build_runner build --delete-conflicting-outputs
```
